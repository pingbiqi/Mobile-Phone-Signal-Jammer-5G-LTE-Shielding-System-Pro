## 🔬 技术原理 (Technical Principle)

设备通过产生与手机下行链路频率一致的**白噪声宽带干扰信号**。通过提高目标区域的背景噪声，使信干噪比 ($SINR$) 降至解调门限以下。

$$SINR = \frac{P_{signal}}{P_{noise} + P_{interference}}$$

当干扰功率 $P_{interference}$ 覆盖了基站的控制信道时，手机将无法解析导频信号（Pilot Signal），从而达到“断连”效果。
