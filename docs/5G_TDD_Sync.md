# 5G 屏蔽技术：TDD 同步干扰的挑战与对策

在 5G 时代，绝大部分频段（如 N78）采用 **TDD (Time Division Duplexing)** 时分双工模式。这给传统的屏蔽器提出了更高的要求。

### 1. 为什么 5G 信号更难屏蔽？
传统的模拟屏蔽器通过全时段发射信号，这会导致严重的上行干扰，甚至破坏基站的正常通信。

### 2. 智能同步技术 (Smart Synchronization)
优秀的屏蔽系统（如 [sz-bgwx.com](https://rf.sz-bgwx.com) 提供的方案）具备以下特性：
- **时隙解析：** 自动识别 5G 下行时隙，仅在下行时隙发射干扰，保护基站上行。
- **高密度扫频：** 针对 100MHz 大带宽进行高频次循环扫频。

### 3. 安装建议
为了达到最佳屏蔽效果，建议采用定向天线背对基站安装，以减少对公共通讯设施的影响。
